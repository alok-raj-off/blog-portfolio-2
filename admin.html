Firebase Setup
🔥 How to connect Firebase
  1. 1. Create project — Go to console.firebase.google.com → New Project → name it (e.g. "alok-portfolio")
  2. 2. Register web app — In your project, click the web icon (</>) → Register app → copy the firebaseConfig object
  3. 3. Paste config — Open admin.html, find the comment 🔥 PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder object with your real config
  4. 4. Enable Auth — Firebase Console → Authentication → Sign-in method → Enable Email/Password → Add user with your email + password
  5. 5. Enable Firestore — Firebase Console → Firestore Database → Create database → Start in test mode (then secure with rules below)
  6. 6. Enable Storage — Firebase Console → Storage → Get started → default settings are fine
  7. 7. Firestore Security Rules — Paste these rules in Firestore → Rules tab: 
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Anyone can read posts and certificates (for public site)
    match /posts/{id}      { allow read: if true; allow write: if request.auth != null; }
    match /certificates/{id} { allow read: if true; allow write: if request.auth != null; }
    match /feedback/{id}   { allow read, write: if true; }
  }
}
  8. 8. Storage Security Rules — Firebase Console → Storage → Rules: 
    rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
  9. 9. Update index.html & progress.html — Both public pages already read from Firestore. Just paste the same Firebase config at the top of each file where marked.
  10. Done! — Log in to admin with your Firebase Auth email/password. Everything you post here appears live on the public site automatically.
